# Quatro Brand Refresh
The FIP, a complete brand refresh of a drink company. The brand I chose was Quatro and I have created 3 responsive pages featuring the required specs for each submission. The pages i've chosen to create and make responsive are the index.html, contact.html and the product.html. Each page tries its best to follow the pre-written design document I've created for my brand.
## Installation
No installation needed, just drag the html files into your preferred browser
